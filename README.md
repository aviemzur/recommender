# recommender
Movies/TV Recommender

## Setup

Create file `API_KEY` and paste your [TMDB API key](https://www.themoviedb.org/settings/api) in it.

Install requirements:
```shell
pip install requirements.txt
```

## Run
```shell
streamlit run recommender.py
```
